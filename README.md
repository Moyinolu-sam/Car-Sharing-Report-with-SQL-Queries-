# Car-Sharing-Report-with-SQL-Queries-

## CarSharing Data Analysis Report (2017)
### Overview

This project analyzes car-sharing demand patterns in 2017 using SQL queries. The dataset was processed and queried using MySQL Workbench.

🛠️ Tools Used
MySQL
Google Sheets (data cleaning)
GitHub (documentation)
📊 Dataset Features

The dataset includes:

timestamp
demand
weather conditions
temperature categories
humidity
windspeed
time attributes (hour, weekday, month, season)
