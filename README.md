# Car-Sharing-Report-with-SQL-Queries-

## Car Sharing Data Analysis Report Year 2017.
### Overview

This project analyzes car sharing demand patterns in the year 2017 using SQL queries. The dataset was processed and queried using MySQL Workbench.

### Tools Used
MySQL
Google Sheets for data cleaning
GitHub for documentation

### Dataset Features

The dataset includes:
ID
Timestamp
Demand
Weather
Temperature categories
Humidity
Windspeed
Hour
Weekday
Month
Season
